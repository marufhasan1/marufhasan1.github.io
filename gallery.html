How to add your photos

Place your images inside new_version/gallery/ and add entries to the photos array in the script below. Each entry supports: 

{
  src:      "gallery/your-photo.jpg",
  thumb:    "gallery/your-photo-thumb.jpg", // optional
  caption:  "A short caption",
  location: "Dhaka, Bangladesh",
  date:     "March 2025",
  category: "travel"  // travel | work | events | life | nature
}
